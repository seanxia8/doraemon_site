## What Belongs Here

| Surface | Purpose |
| --- | --- |
| Physics | Detector and reconstruction background |
| Challenge Rules | Shared structure, scoring, validation, and submissions |
| Metrics | Definitions such as Adjusted Rand Index and panoptic quality |
| Software Stack | Containers, streamers, notebooks, validators, repositories |
| Contribution | How to suggest a challenge or candidate solution |

## What Lives Elsewhere

Challenge pages live under <a href="/challenges">Challenges</a>. Dataset
definitions, schemas, access notes, and visualizations live under
<a href="/data-hub">Data Hub</a>.
